\section{Tools}

To implement the project, I will primarily use the Python ecosystem. The main tools are:

\begin{itemize}[leftmargin=*]
    \item \textbf{Python 3.12.3} as the main programming language.
    \item \textbf{VS Code} with the Python
    \item \textbf{Jupyter Notebooks} for early experimentation and data exploration.
    \item \textbf{OpenStreetMap / Overpass API} to fetch bar locations and tags
          (e.g., \texttt{amenity=bar}, \texttt{pub}, \texttt{nightclub}).
    \item \textbf{Streamlit} to build a simple web-based user interface
          for interacting with the recommendation systemand displaying the result
\end{itemize}

These tools allow efficient data access, processing and visualisation.
